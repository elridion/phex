# Phex
PHP serialized decoder and encoder for Elixir.  
Documentation can be found at [hex](https://hexdocs.pm/phex).
## Installation

The package can be installed by adding `phex` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:phex, "~> 0.2"}
  ]
end
```
